## Image Guidelines

### Resolution
- Logos: 500x500px (PNG with transparency)
- Banners: 1920x1080px or 1920x600px
- Service photos: 800x600px
- Workshop photos: 1200x800px

### Format
- Use **JPG** for photographs (compressed, 85% quality)
- Use **PNG** for logos and graphics with transparency
- Use **SVG** for icons (scalable)

### Optimization
- Compress images using TinyPNG or similar
- Keep file sizes under 500KB for web performance
- Use WebP format for modern browsers when possible

### Naming Convention
- Use lowercase
- Use hyphens instead of spaces
- Be descriptive: `engine-repair-service.jpg` not `img1.jpg`

## Photography Tips

### Workshop Photos
- Good lighting (natural or bright LED)
- Clean, organized workspace
- Show modern equipment
- Include team members working
- Capture before/after shots

### Service Photos
- Close-up shots of work being done
- Show quality tools and parts
- Clean, professional setting
- Action shots (mechanics at work)
